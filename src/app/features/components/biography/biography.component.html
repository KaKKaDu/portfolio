Chapter 1

What was the start?

I begun with simple html and css stuff, just because everything you do is displayed on the screen, and you feel so "cool-programmerable"! Even assuming the fact it doesn't have anything in common with real programming, I'd spent plenty of time on doing dry layouts and beautiful css stuff, cuz it was something so understandable and obvious. But, some time I definitely had to move further.

Chapter 2

Moving to actual programming

There is the important moment of my "career": I finally understood, that buttons have to execute something and links have to... link. So I started diving deep into JavaScript, studying it's specifics and integration with DOM. When I had enough of it, I started refreshing everything with TypeScript on the top. All this took some time, but it gave me the basic knowledge for moving to frameworks.

Chapter 3

The final step

Eventually, we can talk about the stuff I'm working on while doing this portfolio: Angular (specifically 17th version). I begun with basic course of this framework, where I'd got everything I needed to do all I could do without it, but much more optimized and with better user experience. Then I moved to RxJS and other more deep-dive things. And that's where I am: you can see results of my work right there, and also in "Works" part of the website.
